CSS Pixels

Just like you've always wanted!

An easy way to create pure CSS pixel art. But not only is it easy; it's also flexible. Your pixel art is created using ems, so you can change its size as easily as declaring a font size.

(Like this: font-size: 20px; Easy, right?)
